Game Rules:

Normal game

The aim of the game is to visit all the beige squares with the Knight. The knight starts in the upper left and can only do knight movements, as in chess. When a field has been visited, it becomes green.

Blue fields cannot be visited, they are obstacles.

For every square you visit you get points. You also get bonus points at the end of the level, the faster you finish a level the more bonus points you get. You lose when time runs out.

Timesave game

Timesave game is similar to normal game, but you start with very little time. When a level ends, the time you have left is passed on to the next level, so the success of the game depends on how fast you did previous levels. Other than that, it's the same as normal game; when time runs out, you lose.
